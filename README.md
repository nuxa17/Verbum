# Building
Create a new virtual environment with Python 3.11.

- Windows:
```
python -m venv env
env\Scripts\activate.bat
pip install -r requirements.txt
```

- macOS:
```
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

Run bundle.bat for Windows and bundle.command for macOS.
